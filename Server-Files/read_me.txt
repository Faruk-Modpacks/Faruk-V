# Installation

1. Check your java version it should be java 8 other versions may crash
2. Run start-server script corresponding to your os .bat for windows .sh for linux/mac
3. Script will ask you to accept eula type TRUE
4. After 3-4 minutes server will start stop server by typing STOP
5. Your server is ready change settings if you want to
6. You can run server by running start-server script again

Note: you can change ram amount in server-setup-config.yaml change 'maxRam: 5G'

# Updating

1. Copy World, Whitelist,server.properties to somewhere
2. Delete old files
3. Install server with new server files
4. Delete new World, Whitelist,server.properties and copy old World, Whitelist,server.properties
